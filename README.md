# estrada-das-variáveis
Jogo para Projeto Final da disciplina Projeto Integrador

Jogo educacional com objetivo de incentivar o pensamento computacional em alunos do Ensino Fundamental com elementos da gameficação.
É um jogo  do tipo plataforma que consiste em um personagem que passa as fases atravessando obstáculos e inimigos para coletar cristais, acumular pontos e progredir na história.

Controles:
A ou seta - Esquerda
D ou seta - Direita
Espaço ou seta - Pular
E - Interagir com objetos

Disponibilizado para experimentação ou download na plataforma Itch.io, segue o link: https://brewl.itch.io/estrada-das-variaveis
